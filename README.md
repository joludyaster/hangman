## How to run?

### 1. Regular run

> Install needed packages using command below:

```python
pip install -r /path/to/requirements.txt
```

> Run the program:

```python
python run hangman.py
```

### 2. Docker run

Furstly you have to [install](https://www.docker.com/) docker

> Run docker with interactive console

```python
docker run -it hangman
```
